# AI Entertainment
  
 AI Entertainment - это сайт об искусственном интеллекте, написанный на HTML, при помощи CSS и JS. 
 ____________________________________________________________________ 
  
 Запуск: 
  
 Перед запуском распакуйте папку из .zip архива (ВАЖНО! Не изменяйте пути файлов в папке, т.е. не перемещайте).

 Запускайте файл \main\main.html (main.html в папке main) через браузер (**Открыть с помощью** + выбирайте удобный вам браузер).

 Проверьте подключение к интернету!
